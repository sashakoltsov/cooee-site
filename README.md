# Cooee

Coded brand prototype (single-file site) for Cooee, an Australian fragrance house.

- `index.html` — the live site (served by GitHub Pages).
- Append `?edit` to the URL to reveal the design/settings panels.
- `assets/` — fonts, logos, packaging viewer, pack SVGs.

Deployed via GitHub Pages at https://cooee.sashakoltsov.com
